# Online Shop

This is a simple online shop with six pages:

- Home page (index.html)
- Products page (products.html)
- About Us page (about.html)
- Sign In page (sign-in.html)
- Register page (register.html)
- Contact Us page (contact.html)

## Features:
- Responsive design for mobile, tablet, and desktop.
- JavaScript validation for the sign-in and register forms.
- Semantic HTML and CSS for a clean structure and design.
- Interactive elements such as buttons for adding products to the cart.

## How to Run:
1. Clone the repository.
2. Open `index.html` in a browser.
